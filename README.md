# 摩利生活 | 时运财运预测

一个基于原生 JavaScript 开发的 H5 时运财运预测页面，无需后端服务和数据库，纯前端实现。

## 项目简介

这是一个轻量级的运势预测应用，用户输入个人信息后，系统会自动计算生肖和星座，并基于这些信息预测用户的近期时运和财运。项目完全使用原生技术栈开发，适合移动端使用。

## 功能特点

### ✨ 核心功能

- **用户信息收集**
  - 姓名输入
  - 性别选择（男/女）
  - 出生年月选择

- **自动计算**
  - 根据出生年份自动计算生肖（十二生肖）
  - 根据出生月份自动计算星座（十二星座）

- **运势预测**
  - 近期时运预测（百分比显示 + 详细描述）
  - 近期财运预测（百分比显示 + 详细描述）
  - 结果稳定性：同一用户在同一时间多次测算，结果保持一致

## 核心算法说明

### 生肖计算

根据出生年份计算生肖，基于 1900 年（鼠年）作为基准年份，使用模运算确定对应的生肖。

### 星座计算

根据出生月份和日期判断所属星座，支持跨年星座（如摩羯座）的正确计算。

### 运势预测算法

使用哈希函数基于以下信息生成稳定的随机数：
- 用户姓名
- 性别
- 生肖
- 星座
- 出生年月
- 当前日期（精确到天）

**稳定性保证**：同一用户在同一天多次测算，由于使用了相同的种子信息，会得到一致的预测结果。

预测结果分为四个等级：
- **极佳** (80-100%)：运势旺盛
- **良好** (65-79%)：运势平稳向上
- **一般** (45-64%)：运势平稳
- **需注意** (20-44%)：需要谨慎应对
